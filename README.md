# Password Generator

This is a simple code that generates random passwords based on user preferences.

## Features
- User can choose password length
- Option to include uppercase letters
- Option to include numbers
- Option to include symbols
- Password is generated randomly and displayed in the terminal.

## Technologies
- Java 8+ (works on most versions)
- Java `Scanner` for user input
- Java `Random` for generating random characters

## How to Run
1. Clone or download the repository.
2. Open the project in your favorite IDE (IntelliJ IDEA, Eclipse, VS Code, etc.).
3. Compile and run the `PasswordGenerator.java` file.
4. Follow the prompts in the console:
    - Enter the desired password length
    - Choose whether to include uppercase letters (y/n)
    - Choose whether to include numbers (y/n)
    - Choose whether to include symbols (y/n)
5. The generated password will be displayed in the terminal.
